# SSTv3 LLM MD

## Using scrape4ai

To run main.py to regenerate sstv3.md

```terminal
git clone https://github.com/michaelcuneo/sstv3-llm-md
cd sstv3-llm-md
python3 -m venv venv
source venv/bin/activate
python3 main.py
```

To use with an LLM, just download or clone, and upload sstv3.md to the LLM
